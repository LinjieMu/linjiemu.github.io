计算机图形学学习笔记：几何变换

发表于2021-09-16|更新于2021-09-16|计算机图形学
|字数总计:2.3k|阅读时长:11分钟|阅读量:

1. 渲染

  1. 定义：将3D场景投影为2D图像
  2. 阶段：
    1. 几何阶段
      • 将3D模型投影变换到图像平面
      • 决定可见的图元（图形元素）
    2. 光栅阶段
      • 决定可见的片元 (fragment)
      • 决定片元的颜色成为彩色像素
  3. 图元：点、线、三角形等基本几何元素

2. 渲染管线

  1. 定义：渲染计算的流水线

    1. 三维顶点
    2. 顶点处理器（几何阶段）
    3. 裁剪器与元素装配（几何阶段）
    4. 光栅化（光栅阶段）
    5. 片元处理（光栅阶段）
    6. 彩色像素

  2. 两个主要过程：几何+像素

  3. 成像模型：3D场景在虚拟相机（眼睛）下的投影

  4. 各种坐标系

    1. 对象坐标系：对象自身所处的坐标系
    2. 世界坐标系：多个对象共处的坐标系
    3. 视点坐标系：观察者视角为原点的坐标系
    4. 裁剪坐标系
    5. 规范化的设备坐标系
    6. 窗口坐标系

  5. 顶点处理：

    1. 流水线中大部分工作是把对象在一个坐标系中的表示转化为另一个坐标系中的表示
    2. 坐标的每一次变换相当于一次矩阵乘法

  6. 投影变换：正交投影、透视投影

  7. 裁剪：超出可视范围的几何元素需要裁剪掉，不参加后面的
    计算。分为：视景体裁剪、窗口裁剪

  8. 消隐：消除隐藏面

  9. 屏幕映射变换

  10. 光栅化

  11. 着色：让3D物体更有空间感

5. 向量与法向量的变换

平移、旋转、均匀缩放能保持法向量的正确性，而错切不行。我们可以通过下面的公式计算法向量的变换矩阵：

切向量满足变换矩阵关系：

vWS=M vOSv_{WS}=M\ v_{OS} vWS​=M vOS​

如图可知，由垂直关系可知：

nOS⊤vOS=0n_{OS}^\top v_{OS} = 0 nOS⊤​vOS​=0

等价变换可得：

nOS⊤(M−1M)vOS=0n_{OS}^\top(M^{-1}M) v_{OS} = 0 nOS⊤​(M−1M)vOS​=0

由矩阵乘法的结合律可得：

(nOS⊤M−1)(MvOS)=0(n_{OS}^\top M^{-1})(Mv_{OS}) = 0 (nOS⊤​M−1)(MvOS​)=0

化简得

(nOS⊤M−1)vWS=0(n^\top_{OS}M^{-1})v_{WS} = 0 (nOS⊤​M−1)vWS​=0

由此可得

nWS⊤=nOS⊤M−1n^\top_{WS} = n^\top_{OS}M^{-1} nWS⊤​=nOS⊤​M−1

故

nWS=(M−1)⊤nOSn_{WS}=(M^{-1})^\top n_{OS} nWS​=(M−1)⊤nOS​
