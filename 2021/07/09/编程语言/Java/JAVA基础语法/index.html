2. 说明：

  1. Java区分大小写

  2. 关键字public称为访问修饰符：控制程序其他部分对这段代码的访问级别；

  3. class后常跟类名，命名规则为：

    1. 必须以字母开头；
    2. 由字母和数字组成；
    3. 长度上基本没有限制；
    4. 不能使用 java 的保留字；

  4. 类的标准命名规范（大驼峰命名法）：以大写字母开头的名字。如果名字中含有多个单词，则每个单词的首字母必须大写；

  5. 源代码的文件名必须与公共类的名字相同，并以.java作为扩展名；

  6. Java 中的每个句子必须以;结束；

  7. Java中的方法调用格式为：

  1. System.out类的println()方法为打印内容并换行，而print()方法只打印内容。

2. 注释

  1. 行注释界定符为//;
  2. 区块注释界定符为/*和*/的组合；（Java中/* */注释不可以嵌套）

3. 数据类型

1. 分类：

  1. 8种基本数据类型（primitive type）

    1. 整数型：byte、short、int、long
    2. 浮点型：float、double
    3. 字符型：char
    4. 布尔型：boolean

  2. 引用数据类型
    字符串、数组、类、接口、Lambda

2. 整型：

类型 存储空间 范围
byte 1 -128 ~ 127
short 2 -32768 ~ 32767
int 4 -2147483648 ~ 2147483647 （刚过20亿）
long 8 -9223373036854775808 ~ 9223373036854775807
  • Java中，整型的范围与运行Java代码的机器无关；
  • 长整型数值需要加上后缀L或l，如400000000000L;
  • 八进制数加前缀0，二进制数加前缀0B或者0b；
  • Java7 后还可以给数字字面量加下划线_，增加可读性；
  • 注意：Java没有任何无符号（unsigned）形式的long、int、short、byte类型。

3. 浮点类型：

类型 存储空间 范围
float 4 6~7位
double 8 15位
  • float类型的数值有一个后缀F或f;
  • 没有后缀默认为double类型数据，也可在数值后加上D或d；
  • 三个特殊的浮点数值：POSITIVE_INFINITY, NEGATIVE_INFINITY, NaN；
  • 浮点数的存储具有一定的偏差，如果不允许偏差，可以使用BigDecimal类。

4. char类型

  • 内存大小：2Byte；
  • Unicode转义序列会在解析代码之前进行处理；
  • 码点(code point)：指与一个编码表中的某个字符对应的代码值。在Unicode标准中，码点采用十六进制书写，并加上前缀U+；
  • Unicode的码点可以分为17个代码平面。第一个代码平面称为基本多语言平面（basic multilingual plane），包括码点从U+0000到U+FFFF；其余16个平面的码点从U+10000到U+10FFFF，包括辅助字符（supplement character）；
  • 在基本多语言平面中，每个字符用16位表示，通常称为代码单元（code unit）；而辅助字符编码则需要一对连续的代码单元。

5. boolean类型

boolean类型有两个值：true和false；

  1. 注意事项:

    一般不推荐使用强制类型转换，因为可能发生精度损失、数据溢出；
    2. byte/short/char这三种类型都可以发生数学运算；
    3. 类型转换不意味着四舍五入,而是直接舍去后面的部分，四舍五入使用Math.round()方法，参数类型为double时返回值类型为long，参数类型为float时返回值类型为int；
    4. byte/short/char类型若进行了数学运算，则会被首先转化为int类型；
    5. 不同类型进行计算，会先统一转换为大范围进行计算；
